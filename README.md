# Git User Search

### Simple front end for the github api that allows searching for user information and repositories

#### Created using `npx create-react-app`

**Prerequisites**
- NodeJS (npm)

**Packages Used**
- `styled-components`
- `react-spinners`
- `react-icons`

**Setup**
- clone repository
- navigate to folder
- run `npm i`
- for local server: `npm start`
- to build for deployment: `npm build`
